# Local Chat Bot
## Requirements
- Llama LLM models (https://huggingface.co/meta-llama/Llama-2-7b-chat-hf)
## Installation
1. Install the requirement text
<code>pip install -r requirements.txt</code>
2. Download the LLama LLM models on huggingface and put it in the folder
3. Fill the model path variable with your model's path
4. Run this code in command line
<code>chainlit run 4-llama.py</code>
